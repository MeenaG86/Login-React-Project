# 🎬 Login-React-Project (Full Stack)

A Netflix-style login application built using **React (Frontend)** and **Node.js + Express (Backend)**.
This project demonstrates authentication flow, API integration, and a responsive UI.

---

## 🚀 Features

* 🔐 Netflix-style Login UI
* ✅ Form validation (email & password)
* 🔄 API integration using Axios
* 🖥️ Express backend with mock authentication
* ❌ Error handling for invalid login
* ✅ Redirect to Dashboard on success
* 🎨 Responsive design using Tailwind CSS
* 🧩 Component-based architecture (Navbar, MovieRow, MovieCard)

---

## 🛠️ Tech Stack

**Frontend**

* React (Vite)
* Tailwind CSS
* React Router
* Axios

**Backend**

* Node.js
* Express
* CORS

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/MeenaG86/Login-React-Project.git
```

---

### 2️⃣ Setup Frontend

```
cd client
npm install
npm run dev
```

Frontend runs on:

```
http://localhost:5173
```

---

### 3️⃣ Setup Backend

```
cd server
npm install
node index.js
```

Backend runs on:

```
http://localhost:5000
```

## 📌 How it Works

1. User enters email and password
2. Frontend validates input fields
3. Data is sent to backend via API
4. Backend checks credentials (mock data)
5. On success → redirect to Dashboard
6. On failure → show error message
